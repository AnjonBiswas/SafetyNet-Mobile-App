**# 🛡️ SafetyNet – Mobile Security & Monitoring System**

SafetyNet is a full-stack mobile application designed to enhance user safety and system integrity through real-time monitoring, secure communication, and backend-driven data management. The project integrates a mobile client with a backend service to create a scalable safety-focused platform.

---

## 📌 Overview

This project aims to build a **mobile safety ecosystem** where users can interact with a secure system that monitors, manages, and processes safety-related data. It follows a modular architecture with separate mobile and backend components.
